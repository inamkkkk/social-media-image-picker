# Social Media Image Picker

A Flutter project for creating a custom image picker widget similar to social media apps.

## Features

- Select multiple images from the gallery.
- Preview selected images.
- State management using Provider.

## Folder Structure


lib/
├── main.dart
├── screens
│   └── image_picker_screen.dart
├── models
│   └── image_model.dart
└── providers
    └── image_picker_provider.dart


## Usage

1.  Clone the repository.
2.  Run `flutter pub get`.
3.  Run `flutter run`.

## Dependencies

-   provider: ^6.0.0
-   image_picker: ^0.8.0
